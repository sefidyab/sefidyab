# Sefid Yab (سفید یاب) v0.7.3

**Sefid Yab** is a transparency initiative designed to identify and map "White SIM" (Unfiltered Internet) users on social media platforms, specifically X (formerly Twitter).

![Sefid Yab UI](https://img.shields.io/badge/Status-Active-success) ![License](https://img.shields.io/badge/License-MIT-blue)

## What is "White SIM"?
"White SIM" or "Classified Internet" refers to internet access provided to specific individuals or organizations in regions with strict censorship (like Iran) that bypasses filtering restrictions. Sefid Yab aims to democratize this information.

## Features
- **Auto-Detection**: Scans profile metadata for location signals indicating unfiltered access.
- **Background Analysis**: Checks user details silently without disrupting browsing.
- **Public Database**: Aggregates finding into a transparent public directory.
- **Privacy First**: Uses only publicly available data (OSINT).

## Installation
1. Install a Userscript manager (Tampermonkey, Violentmonkey).
2. [**Click Here to Install Script**](https://sefidyab.github.io/sefidyab.user.js)
3. Browse X/Twitter as normal. Badges will appear next to verified accounts.
